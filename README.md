# 配料表过敏源智能分析小程序

## 项目背景

​	基于ORC技术识别商品配料表内容，基于ChatGPT大模型编写prompt实现配料表中潜在过敏源的分析，通过小 程序的形式进行交互与展现

## 使用技术栈

1. ECMAScript 6
2. Vant Weapp
3. 微信小程序开发平台

## 使用

1. 下拉仓库至本地
2. 使用微信开发者工具打开项目文件
3. 运行项目代码
4. 实机运行体验，使用微信扫码体验小程序
